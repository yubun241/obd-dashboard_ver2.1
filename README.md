# 藤井工藝 GT DASH Ver.1

BLE OBD2 対応のリアルタイム車載ダッシュボード PWA です。  
スマートフォンをダッシュボードに設置し、ELM327互換アダプターを介して車両データをリアルタイム表示します。

-----

## 対応環境

- **ブラウザ**: Web Bluetooth API 対応ブラウザ（Chrome for Android / Chrome Desktop）
- **OBD2アダプター**: BLE ELM327互換アダプター（UniCarScan 推奨）
- **車載デプロイ**: Raspberry Pi + Chromium (kiosk mode) 対応

-----

## 主な機能

### ウィジェット（最大5パネル自由配置）

|ウィジェット  |内容                 |
|--------|-------------------|
|GEAR    |ギア段数（自動検出）         |
|SPEED   |車速 km/h            |
|INTAKE  |吸気温度 °C            |
|COOLANT |水温 °C              |
|OIL TEMP|油温 °C              |
|BOOST   |ブースト圧 kg/cm²（ターボ車用）|
|THROTTLE|スロットル開度 %          |
|EST.HP  |推定馬力 PS            |
|G-METER |Gセンサー（慣性球方式）       |
|TIMER   |接続時間タイマー           |

### G-METER — 慣性球方式

G-METER は**慣性球方式**を採用しています。実車のオイル入りアナログGメーターと同じ挙動で、ドットが慣性力の方向（車両加速度と逆向き）に動きます。

|走行状態    |ドットの動き|
|--------|------|
|加速      |B（後方）へ|
|制動（ブレーキ）|F（前方）へ|
|右コーナリング |L（左）へ |
|左コーナリング |R（右）へ |

重力成分はローパスフィルターで動的に除去しており、画面回転（landscape 90°/270°）にも対応しています。  
設定画面からキャリブレーション、表示範囲、アラート閾値、軸反転の調整が可能です。

### RPM ドットバー

上部にカスタムカラーのドットバーを表示します。各ドットの点灯しきい値・色は個別に設定可能。設定した警告RPM以上で全ドット点滅します。

### ブーストバー

MAP センサー（PID 010B）から算出したブースト圧をバー表示。ターボ車 / NA車の切替、MIN/MAX 表示範囲を設定できます。

### ギア自動検出

RPM と車速からギア比を計算し、設定したギア比テーブルと照合してリアルタイムでギア段数を表示します。

## セットアップ

### GitHub Pages デプロイ

```
リポジトリに以下を配置:
  index.html
  styles.css
  app.js
  sw.js
  manifest.json
  icon-192.png
  icon-512.png
```

`manifest.json` の `start_url` はリポジトリのパスに合わせて修正してください。

### PWA インストール

Chrome でページを開き、「ホーム画面に追加」からインストールすると、フルスクリーン + landscape 固定で起動します。

-----

## 設定項目

|項目                 |説明                                  |
|-------------------|------------------------------------|
|表示パネル数             |0〜5パネル（0でRPMのみ）                     |
|OBDレスポンスモード        |Single（標準ELM327）/ Double（UniCarScan）|
|ブースト表示             |ON（ターボ車）/ OFF（NA車）                  |
|Est.HP 係数          |PS/(kPa·rpm) 車種ごとに調整                |
|G表示範囲 / アラート閾値     |G-METER の目盛り端とアラート発光の閾値             |
|Gセンサー軸反転           |設置方向に合わせて横軸・縦軸を個別反転可能               |
|RPMドット数 / 警告RPM    |各ドットの色・しきい値を個別設定                    |
|ギア比 / 最終減速比 / タイヤ外径|ギア自動検出の精度に直結                        |

## Service Worker キャッシュ

オフライン動作・PWA キャッシュは `sw.js` で管理しています。  
ファイルを更新した際は `CACHE_VERSION` のバージョン文字列を上げてください（旧キャッシュが自動削除されます）。

```js
const CACHE_VERSION = 'fujii-kogei-v3-014';
```

-----

## 注意事項

- 走行中の操作は危険です。設定は必ず停車中に行ってください。
- Bluetooth 接続後の設定変更は予期しない切断の原因になる場合があります。接続前にウィジェット設定を完了させてください。
- 本ツールは参考情報の表示を目的としており、車両診断・保安基準への適合を保証するものではありません。
