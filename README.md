## 概要

Android Chrome の **Web Bluetooth API** を利用して BLE ELM327 ドングルと直接通信し、車両の OBD2 データをリアルタイム表示する PWA（Progressive Web App）です。

- アプリインストール・アカウント登録 **不要**
- 初回ロード後はインターネット接続なしで動作
- ホーム画面に追加してフルスクリーンで使用可能
- 全設定はスマートフォンのローカルに保存（外部送信なし）

## 表示項目

| 項目 | PID | 詳細 |
|---|---|---|
| 🔴 RPM LED | `010C` | 12ドット発光（緑→黄→赤）/ 警告 RPM 以上で全点滅 |
| 🔢 RPM 数値 | `010C` | エンジン回転数（rpm）大型イタリック表示 |
| ⚡ ブーストバー | `010B` | MIN〜MAX 範囲を可視化、MIN/MAX/ON-OFF 設定可能 |
| 🚗 車速 | `010D` | 速度（km/h） |
| ⚙️ ギア | 算出 | RPM/速度比からギア段を自動推定 / ギア段で枠色変化 |
| 🌡 水温 | `0105` | 冷却水温（°C）/ **105°C 以上で赤色警告** |
| 🛢 油温 | `015C` | オイル温度（°C）/ **130°C 以上で赤色警告** |

## 設定（CFG）

ステータスバーの **`CFG`** ボタンからすべての設定を変更・保存できます。

### OBD レスポンスモード

| モード | 対象 | 説明 |
|---|---|---|
| `Single` | 標準 ELM327 | 1クエリ = 1レスポンスのみ処理 |
| `Double` | **UniCarScan**（デフォルト） | 重複レスポンスを全処理 |

### 表示設定

| 項目 | 説明 |
|---|---|
| ブースト ON / OFF | NA 車では OFF にしてバーを非表示 |
| ブースト MIN | バー左端の値（例：-0.5 で負圧域も表示） |
| ブースト MAX | バー右端の値（例：1.2 で低ブースト車両に最適化） |

### RPM ドット設定

- ドット数：4〜20個
- 各ドットの**発光色**（green / yellow / red / blue / orange / white）と**点灯開始 RPM 閾値**を個別設定可能
- 警告 RPM 以上で全ドットが点滅

### ギア比 / 駆動系

| 項目 | 説明 |
|---|---|
| 最終減速比 | ファイナルドライブ比 |
| タイヤ外径 | タイヤ総外径（mm） |
| 段数 | 3〜10速まで設定可能 |
| 各段ギア比 | 1速から順に個別入力 |
| 許容誤差 | ギア判定の RPM/速度比の許容範囲（%） |

> 設定変更は保存後すぐに反映されます。設定は `localStorage` に保存され、次回起動時も引き継ぎます。

|---|---|---|---|---|---|

| ドット | 色 | 点灯開始 RPM |

|---|---|

| MAX | `2.0 kg/cm²` |

## 使い方

### 接続手順

1. OBD2 ポートに BLE ELM327 ドングルを挿入
2. エンジン始動（またはイグニッション ON）
3. スマートフォンの Bluetooth を有効化
4. Android Chrome で **https://yubun241.github.io/obd2-dashboard_ver2/** を開く
5. 右下の **`CONNECT`** ボタンをタップ
6. デバイス選択ダイアログでドングルを選択して接続

> **2回目以降:** 「前回のデバイスに接続」ダイアログが自動表示されます。

### PWA としてホーム画面に追加

```
Chrome メニュー（⋮）→「ホーム画面に追加」→ フルスクリーンで起動
```

---

## ギア判定アルゴリズム

```
1. 車速 < 4 km/h または RPM < 500 rpm  →  N（ニュートラル）
2. RPM ÷ 車速 のサンプルを 5個 蓄積
3. 5サンプルの中央値を算出（外れ値・ノイズ除去）
4. サンプル内の最大乖離が 15% 超  →  変速中と判定し前回ギアを保持
5. 各ギア比と中央値を照合、許容誤差 10% 以内を候補に採用
6. 同一ギアが連続 2回 確定  →  ギア表示を更新（ヒステリシス制御）
```

## ポーリング仕様

| 項目 | 値 |
|---|---|
| ポーリング間隔 | 50 ms |
| 優先取得（毎サイクル） | `010C` RPM、`010D` 速度 |
| 間引き取得（順番に） | `015C` 油温、`010B` MAP、`0105` 水温 |
| タイムアウト | 500 ms 無応答で次 PID へ |

## 配布について

このアプリは URL（**https://yubun241.github.io/obd2-dashboard_ver2/**）を共有するだけで、誰でもインストール不要で使用可能です。

- サーバーコスト: **無料**（GitHub Pages）
- 個人情報: **サーバーに送信されない**
- OBD データ: **車両 → スマホの BLE 通信のみ**で外部に出ない
- 設定: **各端末のローカル保存**（共有されない）

> ただしデフォルト設定は Mini F56 JCW のギア比・タイヤ径です。他車種ユーザーは CFG から各車種に合わせて変更してください。

---

## 注意事項

- 本アプリは個人使用目的で作成されています
- 走行中の操作は危険です。**停車中または同乗者による操作**を推奨します
- OBD2 ドングルの品質・互換性により一部 PID が取得できない場合があります
- 更新後にキャッシュが残る場合は Chrome の「サイトデータを消去」をお試しください
